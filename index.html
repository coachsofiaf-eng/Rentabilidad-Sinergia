Simulador de Rentabilidad Multiproveedor

costo → markup → precio → IIBB → rentabilidad

1. Cargar lista de precios

Aceptamos Excel (.xlsx) o CSV. Si el archivo tiene varias hojas, cada hoja se toma como un proveedor distinto. Podés subir varios archivos: se van sumando a la lista (no se pisan entre sí).

2. Parámetros globales

Se aplican a todos los productos. Podés ajustar cada producto individualmente en la tabla.

3. Detalle por producto

Editá markup, descuento o unidades estimadas por mes directamente en la tabla.
Producto Proveedor Costo prov. Desc. % Costo ajust. Markup % Precio s/IVA Precio c/IVA IIBB Rent. bruta Rent. neta Margen Unid./mes
Todavía no cargaste ninguna lista de precios. Subí un archivo arriba para empezar.

Costos fijos mensuales

Cargá el total de costos fijos del mes (alquiler, sueldos, logística, etc. — ya calculado por vos).

Contribución por producto al punto de equilibrio

Contribución mensual = rentabilidad neta unitaria × unidades estimadas por mes. Ordenado de mayor a menor aporte.
Producto Rent. neta unit. Unid./mes Contribución % de costos fijos
Cargá productos con unidades estimadas en la pestaña "Análisis de Precios" para ver este desglose.
